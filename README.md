# Dockerized Java Application

This project demonstrates how to containerize a simple Java application using Docker and run it inside a container.

## Tech Stack
- Java
- Docker
- Linux
- AWS EC2
- Git & GitHub

## Project Architecture

Developer → GitHub → Docker Build → Docker Container → AWS EC2

## Steps to Run

1️⃣ Clone repository

git clone https://github.com/yourusername/simple-java-docker.git

2️⃣ Build Docker image

docker build -t java-app .

3️⃣ Run container

docker run java-app
