# Dockerized Java Application

This project demonstrates how to containerize a simple Java application using Docker and run it inside a container.

## Tech Stack
- Java
- Docker
- Linux
- Git
- GitHub

## Project Structure

```
simple-java-docker
│
├── Dockerfile
├── README.md
└── src
    └── Main.java
```

## Steps to Run

### 1. Clone the repository

```
git clone https://github.com/shreyash005-source/simple-java-docker.git
```

### 2. Navigate to the project folder

```
cd simple-java-docker
```

### 3. Build the Docker image

```
docker build -t java-app .
```

### 4. Run the Docker container

```
docker run java-app
```
